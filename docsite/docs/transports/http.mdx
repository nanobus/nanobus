This component provides extensible HTTP server capabilities.
The functionality of your application is added
by including `routers` and `middleware` components.

### Basic configuration

  values={[

```yaml
transports:
  http:
    uses: "nanobus.transport.http.server/v1"
    with:
      address: ':8080'
      middleware:
        ### See middleware
      routers:
        ### See routers
```

  </TabItem>
  <TabItem value="typescript">

```typescript
import { HttpServerV1 } from "https://deno.land/x/nanobus_config@v0.0.12/mod.ts";

app.transport(
  "http",
  HttpServerV1({
    address: ":8080",
    // See middleware
    middleware: [],
    // See routers
    routers: [],
  }),
);
```
